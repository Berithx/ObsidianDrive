# 1. JWT
## ■ JWT
### - JWT란
- Json Web Token
- JSON 포맷을 이용하여 사용자에 대한 속성을 저장하는 Claim 기반의 Web Token
- 토큰의 한 종류로 쿠키 저장소를 통하여 JWT를 저장된다.
