# 1. 쿠키와 세션
## ■ 쿠키와 세션이란?

### - 쿠키(Cookie)
- 클라이언트에 저장될 목적으로 생성한 작은 정보를 담은 파일
- 구성요소
	- Name : 쿠키를 구별하는데 사용되는 키 (Unique)
	- Value : 쿠키의 값
	- Domain : 쿠키가 저장된 도메인
	- Path : 쿠키가 사용되는 경로
	- Expires : 쿠키의 만료기한 (만료깋)
